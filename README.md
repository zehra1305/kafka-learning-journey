# Kafka Mini Demo

Bu proje, Apache Kafka ve Spring Boot entegrasyonunu öğrenmek için geliştirilmiş temel bir mesajlaşma uygulamasıdır. Web arayüzü üzerinden gönderilen mesajların Kafka üzerinden iletilmesini ve tüketilmesini kapsar.

## Teknolojiler

Java 23 & Spring Boot 4.0.6

Apache Kafka (Docker - 7.4.0)

## Çalıştırma

1.docker-compose up -d ile altyapıyı başlatın.

2.Uygulamayı (Spring Boot) çalıştırın.

3.http://localhost:8080 adresinden mesaj gönderin.
