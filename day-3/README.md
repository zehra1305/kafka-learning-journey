#  Kafka Transfer Uygulaması 

Bu proje, Spring Boot ve Apache Kafka kullanarak geliştirilmiş asenkron bir banka transfer simülasyonudur. Kullanıcılar bir web arayüzü üzerinden transfer isteği gönderir ve bu istek Kafka üzerinden işlenerek tüketici (consumer) tarafından okunur.

## 🛠 Kullanılan Teknolojiler
* **Java 21** & **Spring Boot 3.4.0**
* **Apache Kafka** (Mesaj kuyruğu)
* **Docker & Docker Compose** (Zookeeper ve Kafka kurulumu için)
